Major Projects


  • Support Vector Machine (machine Learning)

    Vector Machine Model that uses a Grid search to optimize gamma and C values for the model. This model works on a built in data set from the seaborn library

  • AI Othello Program

    Python program that allows the user to play the board game Othello. The computer player is backed by an AI system that utilizes Alpha-Beta Pruning and a Mini-Max algorithms to allow the computer to decide which move to make. The board configuration is evaluated using multiple Static Evaluation Functions for different states in the game.

  • Land-Based College Map

    Web based map of the Land-Based Colleges in the US. Also, there is a layer that shows the populations of the world. The map was developed using the python library Folium.
